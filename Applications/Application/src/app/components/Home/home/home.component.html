ABOUT

Our Kanban board application allows employees to register, create multiple projects, and assign themselves to projects. Each employee can be assigned up to 5 projects to ensure a balanced workload. Within projects, employees can assign tasks to each other, create and update tasks, and track their progress through different stages. Employees can delete tasks they have created, and project creators have the ability to permanently delete all tasks associated with the project.
